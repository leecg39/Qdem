# QDEM .cop CSO Deal Pipeline

## 사용 시점

- 고객사 롱리스트를 우선순위화할 때
- 파일럿 계약 가능성을 비교할 때
- 조달/RFP 기회를 선별할 때
- 멈춘 영업 딜을 다시 움직일 때

## 실행 절차

1. 리드를 네 단계로 분류합니다.
- 탐색
- 접촉
- 제안
- 협상

2. 각 리드의 열수요, 탈탄소 압박, 의사결정 속도를 점수화합니다.

## 출력 포맷

```markdown
# CSO 파이프라인 메모 — {날짜}

## 상위 타깃
1.
2.
3.

## 오늘의 영업 액션
- 신규 접촉:
- 제안 업데이트:
- 협상 가속:

## 멈춘 딜 복구안
- 대상:
- 원인:
- 다음 행동:
```
