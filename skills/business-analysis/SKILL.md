# 경영분석 스킬 (Business Analysis Skill)

이 스킬은 CEO가 회사 전반의 경영 상태를 분석하고 전략적 의사결정을 내릴 수 있도록 지원합니다.

## 사용 시점

다음 상황에서 이 스킬을 호출하세요:
- 매주/매월 정기 경영 보고서 작성 시
- 에이전트 성과 및 비용 효율성 검토 시
- 목표(Goal) 달성 현황 파악 시
- 이슈 병목 및 블로커 분석 시
- 전략적 우선순위 재설정 필요 시
- 예산 소진율 이상 감지 시

---

## 분석 절차

### 1단계 — 비용 현황 분석

```
GET $PAPERCLIP_API_URL/companies/{companyId}/costs/summary
GET $PAPERCLIP_API_URL/companies/{companyId}/costs/by-agent
GET $PAPERCLIP_API_URL/companies/{companyId}/costs/by-project
```

분석 항목:
- 총 지출 vs 예산 (소진율 %)
- 에이전트별 비용 효율 (비용 / 완료 이슈 수)
- 전월 대비 증감률
- 80% 초과 에이전트 식별 → 즉시 경보

인사이트 포맷:
```
💰 비용 현황
- 이번 달 총 지출: $X.XX (예산 대비 XX%)
- 최고 비용 에이전트: {이름} ($X.XX)
- 효율 최우수: {이름} (이슈 당 $X.XX)
- ⚠️ 주의: {에이전트} 예산 XX% 소진
```

### 2단계 — 에이전트 성과 분석

```
GET $PAPERCLIP_API_URL/companies/{companyId}/agents
GET $PAPERCLIP_API_URL/companies/{companyId}/issues?assigneeAgentId={id}&status=done
```

각 에이전트별 측정:
- **완료율**: done 이슈 / 전체 할당 이슈
- **평균 처리 시간**: (완료일 - 생성일) 평균
- **블로커 비율**: blocked 이슈 / 전체
- **에러율**: 최근 30일 error 상태 횟수
- **상태**: idle / running / error / paused

성과 등급:
- 🟢 우수: 완료율 80%+, 에러율 10% 미만
- 🟡 보통: 완료율 50-80%
- 🔴 미흡: 완료율 50% 미만 또는 에러율 30%+

### 3단계 — 이슈 파이프라인 분석

```
GET $PAPERCLIP_API_URL/companies/{companyId}/issues?status=todo,in_progress,blocked,done
```

분석 항목:
- **파이프라인 현황**: 상태별 이슈 수
- **병목 식별**: blocked 이슈 목록 및 블로킹 원인
- **우선순위 분포**: critical/high/medium/low 비율
- **미배정 이슈**: assignee 없는 이슈 → CEO가 직접 배정 권고

보고 포맷:
```
📋 이슈 파이프라인
- 할 일: X건 | 진행 중: X건 | 차단됨: X건 | 완료: X건
- 🚨 긴급 처리 필요: {블로커 이슈 목록}
- 미배정 이슈: X건 → 즉시 배정 필요
```

### 4단계 — 목표(Goal) 달성률 분석

```
GET $PAPERCLIP_API_URL/companies/{companyId}/goals
GET $PAPERCLIP_API_URL/goals/{goalId}/issues
```

측정 항목:
- 목표별 연결 이슈 완료율
- 기한(target date) 대비 진행률
- 목표 계층 구조 (상위 목표 → 하위 목표) 달성 흐름

달성률 계산:
```
달성률 = (완료된 연결 이슈 수 / 전체 연결 이슈 수) × 100
```

### 5단계 — 전략적 인사이트 도출

분석 결과를 바탕으로 다음을 판단하세요:

**즉시 조치 (Action Required)**
- 예산 80% 초과 에이전트 → 작업 중단 또는 재배정
- 블로커 이슈 → 원인 파악 후 제거
- 에러 상태 에이전트 → 재시작 또는 설정 점검

**단기 개선 (1-2주)**
- 성과 미흡 에이전트 → 역할 재정의 또는 스킬 추가
- 우선순위 불균형 → 이슈 재정렬
- 목표 달성률 50% 미만 → 하위 목표 분해 또는 리소스 추가

**전략적 검토 (월간)**
- 에이전트 팀 구조 최적화
- 예산 재배분
- 새 목표 설정 또는 기존 목표 수정

## 경영 보고서 작성 포맷

분석 완료 후 Paperclip 이슈에 다음 포맷으로 보고서를 작성하세요:

```markdown
# 경영 분석 보고서 — {날짜}

## 요약 (Executive Summary)
{3줄 이내 핵심 현황}

## 비용 현황
{비용 분석 결과}

## 에이전트 성과
{에이전트별 성과 표}

## 이슈 파이프라인
{파이프라인 현황}

## 목표 달성률
{Goal 달성률}

## 권고 사항
### 즉시 조치
1. ...

### 단기 개선
1. ...

### 전략적 검토
1. ...
```

## KPI 대시보드 기준값

| KPI | 목표 | 경보 기준 |
|-----|------|---------|
| 예산 소진율 | 80% 이하 | 90% 초과 시 경보 |
| 이슈 완료율 | 70% 이상 | 50% 미만 시 경보 |
| 에이전트 가동율 | 80% 이상 | 60% 미만 시 경보 |
| 블로커 비율 | 10% 이하 | 20% 초과 시 경보 |
| 평균 이슈 처리시간 | 3일 이하 | 7일 초과 시 경보 |
