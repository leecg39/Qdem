# KPI 템플릿 및 계산식

## 에이전트 효율성 지수 (AEI)

```
AEI = (완료 이슈 수 × 100) / (비용 $달러 × 운영일수)
```

높을수록 좋음. 동일 역할 에이전트 간 비교에 사용.

## 이슈 속도 (Velocity)

```
주간 속도 = 지난 7일 완료 이슈 수
월간 속도 = 지난 30일 완료 이슈 수
```

## 블로커 영향도

```
블로커 영향도 = blocked 이슈의 priority 가중합
(critical=4, high=3, medium=2, low=1)
```

10 이상이면 즉시 CEO 개입 필요.

## 목표 건강도 (Goal Health Score)

```
건강도 = (완료 이슈 / 전체 이슈) × 0.6 + (남은 기간 / 전체 기간) × 0.4
```
- 0.8 이상: 🟢 건강
- 0.5~0.8: 🟡 주의
- 0.5 미만: 🔴 위험

## 월간 번다운 예측

```
예상 완료일 = 오늘 + (잔여 이슈 수 / 주간 속도 × 7)
```
