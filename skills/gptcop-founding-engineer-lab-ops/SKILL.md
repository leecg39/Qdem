# QDEM .cop Founding Engineer Lab Ops

## 사용 시점

- 실험 시작 전 준비와 안전 점검이 필요할 때
- 측정 데이터 기록 형식을 표준화할 때
- 프로토타입 수정 우선순위를 정할 때
- CTO에게 실험 결과를 간결히 보고할 때

## 실행 절차

1. 실험 전 체크리스트를 완료합니다.
- 환기
- 누출 감지
- 센서 캘리브레이션
- 차단 밸브

2. 실험 중에는 조건/결과/이상징후를 분리 기록합니다.

3. 종료 후 CTO에게 바로 넘길 요약을 만듭니다.

## 출력 포맷

```markdown
# 엔지니어 실험 메모 — {날짜}

## 실험 조건
- 온도:
- 압력:
- 유량:

## 결과
- 열출력:
- 효율:
- 이상징후:

## 다음 수정안
1.
2.
3.
```
